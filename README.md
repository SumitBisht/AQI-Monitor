## Documentation

This uses Arduino IDE to write the script for basic AQI monitoring (the CO2 ppm) locally.
Also, this project aims to upload the aqi data on a realtime basis to a central server for further data caliberation.


## Installation

Install the dependencies first through the Arduino IDE boards/library manager.

```bash
  - esp8266 community
  - MQ135 library
  - Adafruit SSD1306
  - Adafruit SSD1306 wemos mini OLED
  - Adafruit GFX library
```
